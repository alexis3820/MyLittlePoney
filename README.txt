Projet réalisé dans le cadre de la LP dev Web.

On a l'accès à la génération des données en allant sur la route : http://mylittleponey/Generator/home 
Puis appuyer dans l'ordre sur les boutons
- créer la base de données  
- créer les utilisateurs 
- insérer des données après la création de la bdd


Ensuite il faut se login avec les utilisateurs : http://mylittleponey/user/default
utilisateur : ...
mot de passe : ...

On a l'accès au panel : http://mylittleponey/panel/default
Il faut choisir la base de données mylittleponey et on peut voir, modifier et supprimer les données.

On a l'accès au logs : http://mylittleponey/log/show
Où l'on peut voir les bases de données et les utilisateurs. 
On peut aussi afficher les droits des utilisateurs.
On a aussi accès au thread et on peut les tuer.
